# Page CRO Skill

When implementing landing pages, use the following CRO best practices:
1. **Clear Hero CTA:** Place a single, high-contrast primary Call-To-Action above the fold. 
2. **Social Proof:** Integrate testimonial blocks, client logos, and trustworthy statistics early in the scrolling experience.
3. **Frictionless Forms:** If generating leads, keep forms to 2-3 fields minimum (Name, Email, Goal). Make the submit button copy explicitly action-oriented (e.g., "Get My Free Audit" vs "Submit").
4. **Visual Hierarchy:** Guide the user's eye from the Problem statement -> Solution -> Social Proof -> Action. Use whitespace aggressively to avoid cognitive overload.
