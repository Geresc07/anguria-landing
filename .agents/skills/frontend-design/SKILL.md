# Frontend Design Skill

When designing frontend components:
1. **Bold Aesthetic Choices:** Avoid standard Bootstrap/Tailwind off-the-shelf looks. Use premium HSL color palettes or deep dark modes.
2. **Distinctive Typography:** Implement modern Google Fonts (Inter, Roboto, Outfit) instead of default sans-serif. Use high-contrast font weights.
3. **Animations:** Always integrate Framer Motion or pure CSS micro-interactions on hover and scroll (fade-ins, glassmorphism, transform: scale(1.02)).
4. **Context-Aware:** Tailor the frontend aesthetic explicitly to the user's business narrative.
